# Book Cover Images

This folder contains custom book cover images for the e-commerce site.

## Current Images:
- `it-ends-with-us-cover.jpg` - Custom cover for "It Ends with Us" by Colleen Hoover

## Instructions:
1. Save your book cover images in this folder
2. Update the image path in script.js to reference the local file
3. Ensure images are optimized for web (recommended: 400x600px or similar book cover ratio)
