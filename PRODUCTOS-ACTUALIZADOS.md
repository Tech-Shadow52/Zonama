## Ventajas de Usar Imágenes Locales

✓ **Carga más rápida**: No depende de servicios externos
✓ **Mayor confiabilidad**: Las imágenes siempre estarán disponibles
✓ **Sin límites de API**: No hay restricciones de uso
✓ **Mejor rendimiento**: Menor latencia al cargar
✓ **Control total**: Puedes optimizar y modificar las imágenes

## Formato de Imágenes

- **AVIF**: Formato moderno con excelente compresión
- **PNG**: Para imágenes que requieren transparencia
- Ubicación: `productos/[nombre-archivo].avif`

## Compatibilidad

Las imágenes AVIF son compatibles con navegadores modernos:
- Chrome 85+
- Firefox 93+
- Safari 16+
- Edge 85+

Para navegadores antiguos, el sistema mostrará un ícono de placeholder.
